G2 Scraper Assignment Solution

Cleaning the names of the companies on the dataset to use them as the search queries on G2.
Create a scraper of G2 using the name of the companies that search for the first 5 results on the queries and return 'no info' if there's no information about
every company in this platform. The scraper checks that this results have the same website than the one listed on the dataset (because if its gets info about the
first result it's possible that this is not the same company), and if it gets a hit retrieves the information about the G2 website of this company, product description, 
rating, number of reviews and it's alternatives.

It eventually gets blocked by G2 security system.
